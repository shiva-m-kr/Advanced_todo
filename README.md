# TaskFlow - Advanced Todo App

A feature-rich Todo Application built with HTML, CSS, and Vanilla JavaScript.

This project goes beyond basic CRUD functionality by including task prioritization, filtering, sorting, search, statistics tracking, theme switching, and persistent storage using Local Storage.

---

## Live Demo

```md
https://shiva-m-kr.github.io/Advanced_todo/
```

---

## Features

### Task Management

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Reopen completed tasks

### Priority Management

* High Priority Tasks
* Medium Priority Tasks
* Low Priority Tasks
* Visual Priority Indicators

### Important Tasks

* Mark tasks as important
* Dedicated important tasks filter
* Quick identification of critical tasks

### Search

* Real-time task search
* Instant filtering while typing

### Filtering

* All Tasks
* Completed Tasks
* Pending Tasks
* Important Tasks

### Sorting

* Sort by Newest
* Sort by Oldest
* Sort by Priority

### Statistics

Each task includes:

* Creation Time
* Completion Time
* Completion Status
* Priority Information
* Time Taken to Complete

### Theme Support

* Dark Theme
* Light Theme
* Theme preference saved automatically

### Data Persistence

* Tasks stored using Local Storage
* Data remains available after page refresh

### Keyboard Shortcuts

| Key         | Action          |
| ----------- | --------------- |
| Enter       | Add Task        |
| Arrow Up    | High Priority   |
| Arrow Down  | Low Priority    |
| Arrow Left  | Medium Priority |
| Arrow Right | Medium Priority |
